08 — Motion
Animation & Easing
Motion should feel intentional, not decorative. One well-orchestrated page load with staggered reveals creates more delight than scattered micro-interactions. Hover the boxes to feel each curve.

Scroll Reveal Pattern (GSAP + Framer Motion)
All sections: opacity: 0 → 1, y: 28 → 0 on IntersectionObserver trigger at threshold: 0.12. Stagger delay: nth-child × 0.1s. Use var(--ease-out-expo) for all scroll reveals. Hero elements use animation-delay CSS stagger on load. GSAP ScrollTrigger for parallax effects on hero glow orbs only — don't over-use.

1. Add Google Fonts import to index.html or globals.css
2. Paste CSS custom properties into globals.css
3. Extend tailwind.config.js with the color/font/radius config above
4. Find-and-replace all instances of "Helpr" with "Mantle" across the codebase
5. Set body { background: #FAF8F5; font-family: 'Geist'; } as base
6. Never use #000000 or #FFFFFF — always use token values
7. All interactive elements: add transition: all 250ms cubic-bezier(0.19,1,0.22,1)
8. All scroll reveals: opacity:0 → 1, translateY:28px → 0 via IntersectionObserver
